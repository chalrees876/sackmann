About This Project

This web app was built as a personal data science and software engineering project. It uses match statistics from professional tennis players to train a logistic regression model that predicts match outcomes based on serving performance metrics.

The application is built with Django on the backend and Bootstrap for styling. It includes:

  • A searchable database of players and matches.
  • Filterable visualizations showing model performance.
  • Machine learning pipelines implemented with pyspark and scikit-learn and pandas.

This project was developed by Christopher McKenzie as part of his portfolio in Computer Science and Machine Learning. You can explore more projects on GitHub.
